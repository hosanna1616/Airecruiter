# Airecruiter 🚀

![Project screenshot](./public/image.png)

Airecruiter is a modern job recruitment platform built with Next.js and TypeScript. It streamlines the hiring process by providing features for job seekers and employers/government users, including AI-powered screening, profile management, and subscription handling. The application emphasizes security, performance, and a clean user interface.

## Key Features

- **Responsive Next.js frontend** using the App Router.
- **Authentication & authorization** with government, company, and user roles.
- **REST API routes** under `app/api` for various services (jobs, users, screening, etc.).
- **AI integrations** for resume parsing, candidate ranking, and screening.
- **Prisma ORM** for database access with a PostgreSQL schema.
- **File uploads** for resumes, certificates, and documents.
- **Form validation** for user and job data.
- **Internationalization (i18n)** with language context.
